Threema

February 22, 2014Posted by Arturo Argueta

 

Are you a whatsapp user? If you answered yes maybe you are wondering if there are better mobile messaging apps you can use. Threema is a mobile messaging service that provides you more security, because according to Threema all the messages on Threema are encrypted and no one will have access to your messages, except the people you send the messages to. You can also send photos, videos, and several other media with this application. The only bad thing about this application is that it costs 1.99 dollars, but I think that this price is worth it for what you will get in exchange. If you want to get a new way to communicate with your friends choose Threema.

The new OneDrive

January 29, 2014Posted by Arturo Argueta

 

It seems that SkyDrive, a product from Microsoft is changing name. It looks like Skydrive changed name to OneDrive because there was a complaint from the British Sky Broadcasting Group (BSkyB). Microsoft chose OneDrive because there’s going to be just “One” place to store all your files. Users will be able to access files from OneDrive using any mobile device or PC.
I’ve been using OneDrive to sync all my OneNote documents, and I have to say that it is really convenient to have all my notes stored in a backed-up server. It is great to access my notes using any device I want. Also if I’m traveling and I don’t want to take my laptop, but I need to review some notes, I can take a tablet with me and I still have access to all my files and I can still edit them if I need to. Also having Microsoft’s office web applications on OneDrive is something really convenient.
